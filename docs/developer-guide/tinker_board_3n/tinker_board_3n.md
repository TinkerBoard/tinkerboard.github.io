# Tinker Board 3N
## Tinker OS Debian
To build the image, please run the following commands.
```bash
./build.sh rockchip_rk3568_tinker_board_3n_debain_defconfig
VERSION=release ./build.sh
```

## Tinker OS Yocto
To build the image, please run the following commands.
```bash
./build.sh rockchip_rk3568_tinker_board_3n_yocto_defconfig
./build.sh
```
